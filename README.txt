Library Management System

Implementation of a GUI to administrate mySQL
Project about Library Management System that allows the readers (library users) to register themselves to the
library, rent books and return them, without the intervention of the library staff.

The System Hs Two Different Account Types:

Admin – The admin is responsible for adding new titles into the stock and removing
the ones that get out of circulation.
Reader – The reader is the end user of the library, and should be able to sing up to
the library, borrow books and return them.

Mysql Database

The document LibraryManagementSystem.sql include the complete set of instructions to create a complete sample data base. The sample users have already been incluided in this database.
